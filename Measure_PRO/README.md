📏 Measure PRO: Estaqueamento Inteligente (AutoLISP)

⚙️ Como funciona? Em vez de utilizar o comando MEASURE nativo do AutoCAD, que apenas copia blocos idênticos e obriga o usuário a renumerar cada estaca manualmente, este script atua de forma muito mais inteligente:

✨ Principais Vantagens * ⚡ Produtividade Máxima: Distribui, alinha e numera quilômetros de estacas (a cada 20m ou na distância desejada) em questão de segundos.

🛠️ Como Utilizar

Faça o download do arquivo .lsp neste repositório.

Abra o AutoCAD e digite o comando APPLOAD.

Selecione e carregue o arquivo baixado.

Digite o comando MEASUREPRO para iniciar a rotina, selecione seu eixo e siga as instruções no prompt.
